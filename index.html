What

atpackager is a Grunt plugin to create a set of packages from a set of input files, doing transformations in the middle.

Why

atpackager is built in a modular and generic way, allowing you to create advanced packaging configuration and to extend its features with fine-grained control to adjust to the specific needs of your product.
